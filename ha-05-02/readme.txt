# Аналізатор лог-файлів

Цей Python-скрипт призначений для швидкого аналізу текстових лог-файлів. Він дозволяє отримати статистику за рівнями логування (INFO, ERROR, DEBUG, WARNING) та переглянути детальні записи для конкретного рівня.

## Основні можливості
- **Зчитування файлів**: Автоматично парсить дату, час, рівень та повідомлення.
- **Статистика**: Виводить таблицю з кількістю записів кожного рівня.
- **Фільтрація**: Дозволяє вивести повний список повідомлень за конкретним рівнем через аргументи командного рядка.
- **Обробка помилок**: Перевіряє наявність файлу та коректність його формату.

## Вимоги
- Python 3.x
- Лог-файл у форматі: `РРРР-ММ-ДД ГГ:ХХ:СС РІВЕНЬ Повідомлення`

## Використання

### 1. Отримання загальної статистики
Запустіть скрипт, передавши шлях до файлу як перший аргумент:

```bash
python Log-Analyse.py path/to/logfile.log
```

**Приклад виводу:**
```text
Рівень логування | Кількість
-----------------|----------
DEBUG            | 3
ERROR            | 2
INFO             | 4
WARNING          | 1
```

### 2. Перегляд деталей певного рівня
Додайте назву рівня (наприклад, `error`, `info`, `debug`) як другий аргумент:

```bash
python Log-Analyse.py path/to/logfile.log error
```

**Приклад виводу:**
```text
Деталі логів для рівня 'ERROR':
2024-01-22 09:00:45 - Database connection failed.
2024-01-22 11:30:15 - Backup process failed.
```

## Структура коду
- `load_logs()`: завантажує дані з файлу.
- `parse_log_line()`: розбирає кожен рядок у словник.
- `count_logs_by_level()`: рахує кількість записів.
- `filter_logs_by_level()`: відфільтровує записи (використовує спискові вирази).
- `display_log_counts()`: форматує вивід таблиці.
