# Консольний бот-помічник

Простий та зручний інструмент для керування списком контактів через термінал.

## Як запустити
Переконайтеся, що ви знаходитесь у папці з проєктом, та виконайте команду:
```
python bot.py
```

## Список доступних команд

Команди не чутливі до регістру (можна писати як `ADD`, так і `add`).


| Команда | Аргументи | Опис |
| :--- | :--- | :--- |
| **hello** | — | Отримати вітальне повідомлення |
| **add** | `[ім'я] [номер]` | Додати новий контакт або перезаписати існуючий |
| **change** | `[ім'я] [номер]` | Оновити номер телефону для існуючого контакту |
| **phone** | `[ім'я]` | Пошук та виведення номера телефону за ім'ям |
| **all** | — | Вивести список усіх збережених контактів |
| **close** / **exit** | — | Завершити роботу програми |

## Приклади використання
- `add Vitaliy 0991234567` — додасть контакт з іменем Vitaliy.
- `phone Vitaliy` — виведе номер телефону для Vitaliy.
- `all` — покаже всі записи у форматі `Ім'я: Номер`.

---
*Примітка: Дані зберігаються в оперативній пам'яті лише під час роботи програми.*
